GET STARTED

더블클릭 하나면 끝

바이너리 다운로드가 아니라, git으로 받아 실행합니다(자동 업데이트를 위해). 접속하신 OS 기준으로 안내해요.

1
Node.js(LTS) 설치 — nodejs.org (런처가 없으면 안내해 줍니다)
2
저장소 받기
git clone https://github.com/1215kkm/PowerTerminal.git
3
실행 — start.bat 더블클릭start.command 더블클릭 (첫 실행은 우클릭→열기). Claude Code가 없으면 설치할지 물어봅니다.
4
폰에서 쓰려면 — 화면의 QR 버튼을 누르고 경고를 읽은 뒤 스캔.
GitHub에서 받기 → ⬇ ZIP 바로 받기

ZIP은 즉시 받아지지만 자동 업데이트가 안 됩니다 — 계속 쓰실 거면 git clone을 권장해요.

⚠️

보안 — 먼저 읽어주세요

QR·접속 주소를 아는 사람은 그 컴퓨터에서 명령을 실행할 수 있습니다. 믿는 사람에게만 공유하세요. 유출됐다면 PowerTerminal을 종료하면 외부 접속이 끊기고 주소가 폐기됩니다(다음 실행 때 새 주소). 유출 책임은 본인에게 있습니다.
